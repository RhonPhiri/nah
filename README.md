# NAH
# Description
New Apostolic Church Hymnal app is a lightweight, offline hymnal app written in dart (beginner level). Inspired by the [CIS app (Tinashe Mzondiwa)](https://github.com/TinasheMzondiwa?tab=repositories), it includes hymns in various local languages including english.

The aim of the app is not to replace the printed hymnals, but to solve some of the challenges that are currently available which includes;
  * Synchronize old lyrics with new ones
  * Show how useful a hymnal is and the abundant songs it includes
  * Use in night or dark areas
  * Reduce the use of printed programs
  * Minimize sharing of lyrics through social media platforms, pdfs or images
